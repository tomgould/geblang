# Third-Party Notices

Geblang and gebweb redistribute portions of the following third-party
libraries. The full per-dependency licence inventory lives in
`docs/dependency-licenses.md`.

This file is a placeholder. The maintainer assembles the full notice
content (canonical licence texts plus per-licence attribution lists)
by hand from the audit in `docs/dependency-licenses.md`.

When populated, this file is embedded into the `geblang` and `gebweb`
binaries; run `geblang licenses` or `gebweb licenses` to view it.
